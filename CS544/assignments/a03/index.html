Action: Click on a label
Target: Swapping the axis and the lebel with immediate right axis and label with exception of the rightmost axis and label.

b. onMouseOver
Action: Hover mouse over label
Target: Bolding the label and changing the color of all polylines to reflect a color scale designed for that axis.

c. Brushing
Action: Brush over axes
Target: select range of values for each axis
By performing this action, Users can look for patterns showing only the points that fall within the selected range of values between axes.

Question 3

One advantage of parallel coordinates over scatterplot matrices is that parallel coordinates can effectively handle a large number of variables. In a scatterplot matrix, each variable is plotted against every other variable, which can quickly become overwhelming and difficult to interpret as the number of variables increases.
One disadvantage of parallel coordinates compared to scatterplot matrices is that parallel coordinates can be more difficult to read and interpret when the range of the variables differs significantly. In a scatterplot matrix, each variable is plotted on its own scale, which allows for easy comparison between variables.
